# career-path-
AI-powered career coaching agent  that analyzes user interests, identifies skill gaps, and generates personalized career roadmaps with daily actionable guidance.

## 🎯 Features

- 🧠 Career Path Recommendation  
- 🗺️ Personalized Roadmap  
- 📊 Skill Gap Analysis  
- 🎯 Daily Action Plan  
- 💬 Confusion Mode (AI guidance)

---

## ⚙️ How it Works

1. User enters interests and goals  
2. AI analyzes profile  
3. Generates:
   - Best career path  
   - Roadmap  
   - Skills to improve  
   - Daily tasks  

---
